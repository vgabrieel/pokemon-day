## 📖 Descrição:
<p>Uma aplicação web desenvolvida com Node.js, Express, MongoDB e EJS que associa datas a Pokémon utilizando a PokéAPI. O site permite que usuários realizem cadastro e login para descobrir qual Pokémon está relacionado à sua data de nascimento, além de exibir o Pokémon correspondente ao dia atual.</p>

<br>

## 🔒 Segurança:
As senhas dos usuários são protegidas utilizando bcrypt, sendo armazenadas no banco de dados apenas em formato hash, sem exposição da senha original.
